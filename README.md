##  Project Overview
[AI CV Builder is a web application that helps users create professional resumes using artificial intelligence.
Users can enter their information, and the system automatically generates and improves CV content such as summaries and experience descriptions,
with the ability to choose templates and export the resume as a ready-to-use PDF.]
---
## Project Objectives

- Build a web application that helps users create professional CVs بسهولة باستخدام الذكاء الاصطناعي  
- Improve CV quality by generating summaries and enhancing experience descriptions  
- Provide ATS-friendly CV templates لزيادة فرص القبول في الوظائف  
- Allow users to manage, edit, and export their CVs بسهولة (PDF)
  
## Project Scope

- User authentication system (Register / Login / Logout)  
- Dashboard to manage CVs (Create, Edit, Delete, View)  
- CV Builder with multi-step form (Personal Info, Experience, Skills)  
- AI features (Generate summary, improve content)  
- Multiple CV templates and export to PDF  
- Light/Dark mode and Arabic/English language support  

> **Final Deliverable:**  
A full-stack AI-powered CV Builder web application that allows users to create, customize, and download professional CVs using modern UI and AI assistance.

## Project Plan (5 Weeks)

### Week 1 — Setup & UI Design
- Setup React project with Vite  
- Install Tailwind CSS and FontAwesome  
- Design main pages (Home, About, Contact)  
- Create basic layout (Navbar, Sidebar)

### Week 2 — Authentication & Routing
- Build Login and Register صفحات  
- Setup React Router  
- Create protected routes (Dashboard access)  
- Connect frontend with backend authentication

### Week 3 — Dashboard & CV Management
- Build Dashboard UI  
- Display user CVs  
- Implement Create, Edit, Delete CV features  
- Setup backend APIs for CV data

### Week 4 — CV Builder & AI Integration
- Create multi-step CV builder form  
- Add AI features (generate summary, improve content)  
- Build CV preview page  
- Implement template switching

### Week 5 — Finalization & Deployment
- Add Dark/Light mode  
- Add Arabic/English translation  
- Fix bugs and improve UI  
- Export CV to PDF  
- Deploy project (Frontend + Backend)
